## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript for building the backend API for an E-commerce website following MVC developement pattern. 

Database: MySQL

ORM: TypeORM

Frontend: React

Implemented features such as interceptor logging, authentication guards, pipes for data transformation available with NestJS. 


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
